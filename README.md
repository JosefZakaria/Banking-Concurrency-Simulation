# Banking-Concurrency-Simulation
A Java application demonstrating core concurrency concepts through a simulated banking system. The project showcases race conditions, synchronized operations using locks, and deadlock prevention via consistent lock ordering.

**Features**

Simulation of concurrent bank-account transfers

Race condition demonstration

Thread-safe operations using synchronized blocks and locks

Deadlock prevention strategy

Clean, simple, object-oriented structure

**Technologies** 

Java

Multithreading

Locks & synchronization

Concurrency control
